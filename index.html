HTML

  • The head element contains information about the webpage.
  • The body element repesents the visible content shown to the user.
  • Elements within other elements are called child elements.
  • The body element is an semantic HTML element, which is an element that outlines its content and it's purpose on the webpage.
  • HTMl element content is in between an opening and a closing tag which tells the browser what to do with the content.
  • When it comes to unordered lists and ordered lists, they should contain list items.
  • The closer a heading element is to 1, the bigger it will be.
  • The src attribute tells where the image is, whether it is on the internet or your local comupter
  • The anchor links the webpage while the href attribute leads to the URL for when the webpage is linked

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations are made up of the CSS property and the value of said property.
  • The selector gives meaning to the element/attributes so that the declarations can be applied.
  • In order to see the styles in the stylesheet on the wepage, you must first link the stylesheet to the HTML file.
  • A class attribute is a way for a CSS rule to be applied to an element of choice.
